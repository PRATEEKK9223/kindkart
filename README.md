# KindKart (Node + Express + EJS + MongoDB)

## Prerequisites
- Node.js 18+ and npm
- MongoDB running locally or a MongoDB connection URI
- (Optional) SMTP credentials for notifications
- (Optional) Map API key

## Setup
1. Clone files into a directory `kindkart`.
2. `cd kindkart`
3. Create `.env` from `.env.example` and fill required values (MONGODB_URI, SESSION_SECRET). Add optional SMTP / MAP keys if needed.
